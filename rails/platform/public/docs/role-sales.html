営業担当の主な業務

営業担当は、案件の獲得から受注確定までを担当します。

案件登録

顧客からの引合いを案件として登録し、基本情報を入力します。

見積作成

工事内容に基づいた見積書を作成し、顧客に提出します。

4点チェック

受注確定のための4つのチェック項目を管理します。

顧客管理

顧客情報の登録・更新を行います。

新規案件の登録

登録手順

  1. 「案件一覧」→「新規案件」をクリック 案件登録フォームが表示されます。
  2. 顧客を選択 既存顧客を選択、または新規顧客を登録します。
  3. 案件名を入力 工事名称を入力します（例：〇〇ビル外壁改修工事）
  4. 現場情報を入力 現場住所、工期予定などを入力します。
  5. 担当者を設定 自分を営業担当として設定します。
  6. 「登録」ボタンで保存 案件コードが自動採番されます。
案件コードについて

案件コードは「PJ + 年月 + 連番」で自動採番されます（例：PJ202601001）。このコードで案件を管理します。

見積書の作成

見積の構成

  • カテゴリ - 工種別にグループ化（外壁工事、足場工事など）
  • 明細項目 - 品名、仕様、数量、単価を入力
  • 諸経費 - 現場管理費などの上乗せ
  • 消費税 - 自動計算（10%）

作成手順

  1. 案件詳細画面で「見積」タブを選択 「新規見積」ボタンをクリックします。
  2. 基本情報を入力 有効期限、宛先（顧客名）を確認します。
  3. カテゴリを追加 「カテゴリ追加」で工種を追加します。
  4. 明細項目を入力 品名・仕様・数量・単価を入力します。
  5. 諸経費率を設定 必要に応じて諸経費率（10%等）を設定します。
  6. プレビュー確認 見積金額を確認し、必要に応じて調整します。
  7. PDF出力 見積書をPDFで出力し、顧客に提出します。

見積のコツ

適正な粗利確保

会社の目標粗利率（通常20〜30%）を確保できる金額設定を心がけます。

競合との差別化

価格だけでなく、品質・工期・アフターサービスも訴求ポイントに。

4点チェック（受注確定）

見積承認後、受注確定のために以下の4項目をチェックします。

1. 契約書

契約書または基本契約の締結確認

  • 契約書の取り交わし
  • 契約内容の確認

2. 注文書

正式な注文書の受領

  • 注文書の受領確認
  • 金額・条件の照合

3. 支払条件

支払条件の確認・合意

  • 支払サイトの確認
  • 資金繰りへの影響確認

4. 顧客承認

顧客側決裁者の承認

  • 決裁権限者の確認
  • 着工許可の取得
4点すべて完了で受注確定

4つのチェックがすべて完了すると、案件ステータスが「受注済」に変更され、工務担当に引き継がれます。

口頭先行フローの場合

急ぎの案件で、口頭で受注を受けてから後で注文書を受領するケースです。

操作手順

  1. 案件登録時に「口頭先行」を選択 受注フローを「口頭先行」に設定します。
  2. 口頭受注を記録 口頭受注金額と日時を記録します。
  3. 着工を進める 注文書なしでも着工できます。
  4. 後日、注文書を受領 注文書を受領したら「注文書受領」ボタンで記録します。
注意：14日ルール

口頭受注から14日を超えて注文書が未受領の場合、アラートが表示されます。早めの注文書取得を心がけてください。

顧客の登録・管理

新規顧客の登録

  1. 「マスタ管理」→「顧客」を選択 顧客一覧画面が表示されます。
  2. 「新規顧客」ボタンをクリック 顧客登録フォームが表示されます。
  3. 顧客情報を入力 会社名、担当者、連絡先を入力します。
  4. 与信限度額を設定（推奨） 新規取引先は与信限度額の設定を推奨します。
  5. 「登録」ボタンで保存 顧客コードが自動採番されます。

登録すべき情報

項目 重要度 用途
会社名 必須 見積書・請求書の宛先
担当者名・連絡先 重要 日常の連絡に使用
住所 重要 書類の送付先
支払条件 重要 請求・入金管理に使用
与信限度額 推奨 債権管理のアラートに使用

日常の確認事項

毎日確認すること

  • 案件一覧で進行中案件のステータス確認
  • 見積の承認状況確認
  • 4点チェックの進捗確認

週次で確認すること

  • 見積中案件のフォローアップ
  • 口頭受注案件の注文書フォロー
  • 新規引合いの案件登録漏れチェック
