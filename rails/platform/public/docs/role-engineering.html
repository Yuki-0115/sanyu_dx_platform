工務担当マニュアル

実行予算・原価管理・現場管理の業務ガイド

工務担当の主な業務

工務担当は、受注後の実行予算作成から施工完了までの原価管理を担当します。

受注確定
→
実行予算作成
→
着工準備
→
日報確認
→
原価管理
→
完工

実行予算作成

受注金額に対する原価計画を立て、目標粗利を設定します。

原価管理

日報データを基に実績原価を把握し、予算との差異を管理します。

現場台帳管理

案件ごとの原価集計、粗利状況を確認します。

協力会社管理

外注先の登録、人工単価の設定を行います。

実行予算の作成

受注確定後、工事を開始する前に実行予算を作成します。

予算項目

項目 内容 入力のポイント
労務費 自社社員の人件費 想定人工数 × 単価で計算
材料費 資材・材料の購入費 見積の材料費を参考に
外注費 協力会社への支払い 人工払い + 請負金額
経費 その他経費 運搬費、消耗品など

作成手順

  1. 案件詳細画面で「実行予算」タブを選択 実行予算画面が表示されます。
  2. 各原価項目を入力 労務費、材料費、外注費、経費を入力します。
  3. 人工単価を設定 正社員単価、臨時社員単価、外注単価を設定します。
  4. 目標粗利率を確認 受注金額に対する粗利率を確認します。
  5. 「予算確定」ボタンで確定 確定後、日報入力が可能になります。

人工単価の目安

区分 デフォルト単価 備考
正社員 ¥18,000/人日 社員の平均人件費から算出
臨時社員 ¥15,000/人日 日給＋社会保険按分
外注 ¥20,000/人日 協力会社との取り決めによる
予算確定後の変更

予算確定後に大幅な変更が必要な場合は、管理者に相談してください。履歴として記録されます。

着工前5点ゲート

着工前に確認すべき5つのチェック項目です。

1. 現場条件確認

現場の状況、搬入経路、電源・水道の確認

2. 夜間作業確認

夜間作業の有無、必要な届出の確認

3. 規制確認

道路使用許可、近隣への告知の確認

4. 安全書類確認

安全書類（グリーンファイル）の準備確認

5. 納品確認

材料・機材の手配・納品予定の確認

5点すべて完了で着工可能

5つのチェックがすべて完了すると、着工準備完了となり、施工を開始できます。

日報の確認・管理

職長が入力した日報を確認し、原価を把握します。

確認ポイント

確認項目 チェック内容
出面（人工） 作業員数が適正か、過剰投入はないか
作業内容 予定通りの進捗か、遅れはないか
材料費 想定内の金額か、予算超過はないか
外注作業 外注人工数が適正か

日報一覧での確認

  1. 案件詳細画面で「日報」タブを選択 日報一覧が表示されます。
  2. 日付順に確認 各日報の内容を確認します。
  3. 問題があれば職長に確認 不明点や異常値があれば、職長に確認します。

現場台帳での原価確認

現場台帳では、案件の原価集計と予算対比を確認できます。

表示内容

項目 予算 実績 差異
労務費 予算で設定した金額 日報から自動集計 自動計算
材料費 予算で設定した金額 日報から自動集計 自動計算
外注費 予算で設定した金額 日報から自動集計 自動計算
経費 予算で設定した金額 日報から自動集計 自動計算
合計 予算原価合計 実績原価合計 差異合計

アラート

予算超過アラート

実績原価が予算を超過すると、現場台帳にアラートが表示されます。早めに対策を検討してください。

協力会社の管理

外注先の協力会社情報を登録・管理します。

登録項目

項目 内容
会社名 協力会社の正式名称
代表者・担当者 連絡先となる担当者
標準人工単価 日報入力時のデフォルト単価
仮社員有無 仮社員がいる場合はチェック

完工処理

工事が完了したら、完工処理を行います。

  1. すべての日報が確定済みか確認 未確定の日報がないか確認します。
  2. 最終原価を確認 現場台帳で最終的な原価を確認します。
  3. 完工日を記録 案件の完工日を入力します。
  4. ステータスを「完工」に変更 これにより、経理担当が請求処理を開始できます。

日常の確認事項

毎日確認すること

  • 進行中案件の日報入力状況
  • 予算超過アラートの有無
  • 完工間近の案件の進捗

週次で確認すること

  • 全案件の現場台帳チェック（予算対比）
  • 協力会社への発注状況確認
  • 材料・機材の手配状況
