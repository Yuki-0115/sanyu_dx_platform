作業員マニュアル

現場作業・安全管理・情報確認ガイド

作業員ができること

作業員は、自分の勤怠情報や案件情報を確認できます。

勤怠確認

自分がどの現場に何日出勤したか確認できます

案件情報

担当している現場の情報を確認できます

安全書類

現場の安全書類を確認できます

日報は職長が入力します

日報（出面・経費など）は職長がまとめて入力します。自分で入力する必要はありません。

勤怠の確認方法

自分の出勤記録を確認できます。

  1. 「マイページ」をクリック 画面上部のメニューから「マイページ」を選びます。
  2. 「勤怠履歴」を確認 いつ、どの現場に出勤したかが一覧で表示されます。
  3. 月を選択して確認 月を選ぶと、その月の出勤日数が確認できます。

表示される情報

項目 内容
日付 出勤した日
現場名 作業した現場の名前
勤務区分 終日（1日）または 半日（0.5日）
人工数 終日＝1.0、半日＝0.5

記録が間違っている場合

出勤記録に間違いがあれば、職長または事務所に連絡してください。

現場情報の確認

担当している現場の情報を確認できます。

  1. 「案件一覧」をクリック メニューから「案件一覧」を選びます。
  2. 現場を選択 確認したい現場をクリックします。
  3. 現場情報を確認 現場名、住所、担当者などが表示されます。

確認できる情報

現場名

工事の名称

現場住所

作業場所の住所

顧客名

発注元の会社名

担当者

営業担当・施工担当の名前

安全について

現場での安全は最も大切です。

安全第一

  • 作業前に必ずKY（危険予知）活動を行う
  • 保護具（ヘルメット、安全帯など）を必ず着用
  • 危険を感じたらすぐに作業を中止して報告
  • 体調不良の場合は無理をせず申告

安全書類の確認

現場の安全書類（グリーンファイル）を確認できます。

  1. 案件詳細から「安全書類」タブを選択 安全書類の一覧が表示されます。
  2. 書類を確認 必要な書類をクリックして内容を確認します。

現場でのトラブル

職長に報告

怪我・事故

すぐに職長・事務所に報告

Worker Web について

スマートフォンからも情報を確認できます。

Worker Web でできること

  • 自分の勤怠履歴の確認
  • 今月の出勤日数の確認
  • 担当現場の情報確認

アクセス方法

  1. スマートフォンでアクセス 案内されたURLにアクセスします。
  2. ログイン 社員番号とパスワードでログインします。
  3. 情報を確認 勤怠や現場情報を確認します。

パスワードを忘れた場合

事務所に連絡して、パスワードのリセットを依頼してください。
