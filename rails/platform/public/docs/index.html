概要 業務フロー マスタ管理 案件管理 見積作成 4点チェック 実行予算 日報入力 請求管理 入金管理 月次処理 経営ダッシュボード

システム概要

SanyuTech DX Platformは、建設会社の業務をデジタル化し、案件の受注から入金までを一元管理するシステムです。

案件一元管理

案件ID をキーに、見積から入金まで全データを紐付けて管理します。

日報による原価管理

職長が一括入力する日報から、リアルタイムで原価を把握できます。

4点チェック

受注前の重要チェック項目を管理し、リスクを未然に防ぎます。

経営ダッシュボード

売上・粗利・資金繰りをリアルタイムで可視化します。

仮社員相殺

協力会社への支払いと売上を自動計算し、相殺処理を効率化します。

Google Drive連携

書類を自動でクラウドにバックアップし、いつでもアクセス可能です。

マスタ管理

システムの基本データを管理します。案件登録の前に、必要なマスタデータを登録してください。

顧客マスタ

発注元の顧客情報を管理します。

項目 必須 説明
顧客名◯会社名または個人名
顧客コード-自動採番（CL + 連番）
担当者名-主要連絡先の担当者
電話番号-連絡先電話番号
メールアドレス-連絡先メール
住所-請求書送付先等
与信限度額-取引上限金額
支払条件-月末締め翌月払い等

協力会社マスタ

外注先・協力会社の情報を管理します。仮社員がいる場合は相殺設定も行います。

項目 必須 説明
会社名◯協力会社名
代表者名-代表者または担当者
仮社員有無-仮社員がいる場合はチェック
相殺ルール-相殺計算のルール設定
人工単価-標準の人工単価

社員マスタ

自社社員の情報を管理します。日報の出面入力で使用します。

項目 必須 説明
社員名◯氏名
社員番号-自動採番
雇用形態◯正社員 / 臨時社員
役職-職長 / 作業員 等
部署-所属部署
権限◯管理者 / 経営 / 営業 / 施工 等
権限について

権限によってアクセスできる機能が異なります。経営ダッシュボードは「経営」「管理者」権限のみ閲覧可能です。

案件管理

新規案件の登録から完了までを管理します。すべてのデータは案件に紐づきます。

案件の登録

  1. メニューから「案件一覧」を選択 案件一覧画面が表示されます。
  2. 「新規案件」ボタンをクリック 案件登録フォームが表示されます。
  3. 基本情報を入力 案件名、顧客、現場住所、担当者などを入力します。
  4. 「登録」ボタンで保存 案件コードが自動採番され、案件が作成されます。

案件種別

種別 説明 用途
通常案件 見積→受注→施工の標準フロー 一般的な工事案件
その他（小工事・常用） 簡略化されたフロー 小規模工事、常用作業

受注フロー

フロー 説明
通常（見積→注文書→受注） 標準的な受注プロセス。注文書受領後に受注確定。
口頭先行（口頭受注→後から注文書） 口頭で受注後、後日注文書を受領するパターン。

見積作成

案件に対する見積書を作成します。カテゴリ別に見積項目を整理できます。

見積書の作成手順

  1. 案件詳細画面から「見積作成」をクリック 新規見積フォームが表示されます。
  2. 見積基本情報を入力 見積番号（自動採番）、有効期限、宛先などを入力します。
  3. カテゴリを追加 「外壁工事」「足場工事」など、工種別にカテゴリを作成します。
  4. 見積項目を追加 各カテゴリに、品名・仕様・数量・単価を入力します。
  5. 諸経費・福利厚生費を設定 カテゴリ別に諸経費率・福利厚生費率を設定します。
  6. プレビュー・PDF出力 見積書をプレビューし、PDF形式で出力します。

見積ステータス

ステータス 説明
draft下書き - 編集中
submitted提出済 - 顧客に提出済み
approved承認済 - 顧客から承認
rejected却下 - 不採用
見積の改訂について

見積を修正する場合は、新しいバージョンを作成することを推奨します。履歴として以前の見積も保存されます。

4点チェック（受注確定）

受注前に確認すべき4つの重要項目をチェックします。すべて完了すると受注確定となります。

4点チェック項目

1. 契約書

契約書または注文書の締結確認。双方の合意内容を書面化。

2. 注文書

顧客からの正式な注文書の受領確認。金額・条件を確認。

3. 支払条件

支払いサイト、締め日、支払日の確認。資金繰りへの影響を把握。

4. 顧客承認

顧客側決裁者の承認確認。着工許可の取得。

チェック手順

  1. 案件詳細画面で「4点チェック」タブを選択 4点チェック画面が表示されます。
  2. 各項目を確認してチェック 確認できた項目にチェックを入れます。必要に応じてメモを記録。
  3. 4項目すべて完了で受注確定 すべてのチェックが完了すると、ステータスが「受注済」に変更されます。
口頭先行フローの場合

口頭で先に受注した場合は、注文書の受領を後から記録できます。未受領の場合はアラートが表示されます。

実行予算

受注確定後、実行予算を作成します。原価管理の基準となる重要なデータです。

予算項目

項目 説明 計算方法
労務費 自社社員の人件費 人工数 × 人工単価
材料費 資材・材料の購入費 見積もり金額
外注費 協力会社への支払い 人工数 × 単価 + 請負金額
経費 その他経費（運搬費等） 見積もり金額

人工単価の設定

予算作成時に、雇用形態別の人工単価を設定します。

雇用形態 デフォルト単価 説明
正社員¥18,000/人日自社正社員の単価
臨時社員¥15,000/人日臨時雇用の単価
外注¥20,000/人日協力会社への支払単価

予算確定

  1. 各項目の金額を入力 労務費、材料費、外注費、経費を入力します。
  2. 目標粗利率を設定 目標とする粗利率（%）を設定します。
  3. 「予算確定」ボタンをクリック 予算が確定され、日報入力が可能になります。
予算と実績の比較

日報入力が進むと、予算と実績の差異がリアルタイムで表示されます。予算超過時はアラートが出ます。

日報入力

現場の作業内容、出面（勤怠）、経費を日次で記録します。職長が一括入力します。

日報の構成

基本情報

日付、天候、作業内容、安全事項などを記録

出面（勤怠）

作業員ごとの出勤状況（終日/半日）を記録

経費

材料費、燃料費、高速代などを記録

外注作業

協力会社の作業内容・人工数を記録

入力手順

  1. 案件詳細画面から「日報」タブを選択 日報一覧が表示されます。
  2. 「新規日報」ボタンをクリック 日報入力フォームが表示されます。
  3. 基本情報を入力 日付、天候、作業内容を入力します。
  4. 出面を入力 「出面追加」で作業員を追加し、勤務時間を入力します。
  5. 経費を入力 材料費、燃料費などを入力します。レシートは添付可能。
  6. 外注を入力 協力会社の作業がある場合は、人工数または請負金額を入力。
  7. 「確定」ボタンで保存 日報が確定され、原価に反映されます。

出面の種類

種類 人工数 説明
終日1.0通常の1日勤務
半日0.5午前または午後のみ
日報の修正について

確定済みの日報を修正する場合は「修正」ボタンから行います。修正履歴は監査ログに記録されます。

請求管理

工事完了後（または出来高に応じて）、請求書を発行します。

請求書の作成

  1. 案件詳細画面から「請求」タブを選択 請求一覧が表示されます。
  2. 「新規請求書」ボタンをクリック 請求書作成フォームが表示されます。
  3. 請求情報を入力 請求金額、消費税、支払期日などを入力します。
  4. 請求明細を追加 見積項目を参照して明細を追加できます。
  5. プレビュー・発行 請求書をプレビューし、「発行」ボタンでステータスを更新。

請求パターン

パターン 説明 用途
一括請求 工事完了後に全額請求 小規模工事
出来高請求 進捗に応じて分割請求 大規模・長期工事
着手金+残金 着手時と完了時の2回 一般的な中規模工事

入金管理

請求に対する入金を記録し、債権管理を行います。

入金の記録

  1. 請求詳細画面から「入金登録」をクリック 入金登録フォームが表示されます。
  2. 入金情報を入力 入金日、入金額を入力します。
  3. 「登録」ボタンで保存 入金が記録され、請求残高が更新されます。

入金ステータス

状態 説明
未入金請求済みだが入金なし
一部入金請求額の一部が入金済み
入金完了全額入金済み
入金遅延支払期日を過ぎて未入金
入金遅延アラート

支払期日を過ぎた未入金請求は、ダッシュボードにアラート表示されます。

月次処理

月次での帳票作成、仮社員相殺処理を行います。

月次出来高

各案件の月ごとの出来高（進捗金額）を記録します。

  1. 案件詳細から「月次出来高」を選択 月次出来高一覧が表示されます。
  2. 対象月と出来高金額を入力 その月に計上する出来高金額を入力します。
  3. 登録 出来高が記録され、仕掛かり管理に反映されます。

仮社員相殺

協力会社から派遣された仮社員について、給与と売上を相殺計算します。

支払側（給与）

仮社員の給与 + 社会保険料を計算

売上側（人工売上）

仮社員の人工数 × 単価で売上を計算

相殺金額

支払額と売上額を相殺した差額

相殺処理手順

  1. 「月次処理」→「仮社員相殺」を選択 相殺対象の協力会社一覧が表示されます。
  2. 対象月・協力会社を選択 相殺計算対象を選択します。
  3. 給与・社会保険を入力 支払側の金額を入力します。
  4. 人工数を確認 日報から集計された人工数を確認します。
  5. 相殺確定 差額が計算され、相殺処理が完了します。

経営ダッシュボード

経営状況をリアルタイムで可視化します。経営層・管理者のみアクセス可能です。

粗利管理

資金繰り

入金予定、支払予定、キャッシュフロー

仕掛かり

債権管理

未入金一覧、入金遅延アラート

案件状況

ステータス別の案件数、進行中案件

アラート機能

アラート 条件 対応
入金遅延 支払期日超過 顧客への督促
