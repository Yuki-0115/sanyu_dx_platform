経営ダッシュボード

経営状況のリアルタイム可視化

経営ダッシュボードとは

経営ダッシュボードは、会社の経営状況をリアルタイムで可視化する機能です。売上・粗利・資金繰り・案件状況など、意思決定に必要な情報を一画面で確認できます。

アクセス権限

経営ダッシュボードは「admin（管理者）」「management（経営）」権限を持つユーザーのみアクセス可能です。

ダッシュボード表示項目

売上サマリー

  • 月次売上（請求ベース）
  • 年度累計売上
  • 前年同月比
  • 月別推移グラフ

粗利管理

  • 月次粗利額・粗利率
  • 案件別粗利ランキング
  • 目標対比
  • 粗利率低下アラート

資金繰り

  • 入金予定（今月・来月）
  • 支払予定（外注・経費）
  • 月末資金残高予測
  • キャッシュフローグラフ

仕掛かり管理

  • 仕掛かり総額
  • 案件別仕掛かり
  • 未請求出来高
  • 進捗率別一覧

債権管理

  • 未入金総額
  • 入金遅延アラート
  • 顧客別残高
  • 回収率

案件状況

  • ステータス別件数
  • 進行中案件一覧
  • 受注予定案件
  • 完工予定案件

売上サマリー

表示内容

指標 計算方法 更新タイミング
月次売上 当月発行請求書の合計（税抜） 請求書発行時
年度累計 年度開始からの売上合計 請求書発行時
前年同月比 当月売上 ÷ 前年同月売上 × 100 月次
月別推移 直近12ヶ月の月別売上グラフ リアルタイム

活用方法

  • 月次の売上目標達成状況の確認
  • 季節変動の把握と対策立案
  • 前年比較による成長率の確認

粗利管理

粗利の計算

粗利額 = 受注金額 - 実績原価 粗利率 = 粗利額 ÷ 受注金額 × 100 実績原価 = 労務費 + 材料費 + 外注費 + 経費

アラート条件

アラート 条件 対応
粗利率低下 粗利率 < 目標粗利率 - 5% 原価分析・対策検討
粗利赤字 粗利額 < 0 緊急対応・原因究明
予算超過 実績原価 > 予算原価 追加費用の精査

資金繰り

入金予定と支払予定から、資金繰りの見通しを確認できます。

入金予定

期間 内容
今週入金予定 支払期日が今週の請求書
今月入金予定 支払期日が今月の請求書
来月入金予定 支払期日が来月の請求書

支払予定

項目 内容
外注支払 協力会社への支払い（相殺後）
経費支払 掛け払い経費の支払い
固定費 家賃、リース料等の固定費
給与 月次給与の支払い

仕掛かり管理

施工中で未請求の出来高（仕掛かり）を管理します。

仕掛かりの計算

仕掛かり金額 = 累計出来高 - 累計請求額 累計出来高 = 月次出来高の合計 累計請求額 = 発行済み請求書の合計

表示内容

  • 仕掛かり総額（全案件合計）
  • 案件別仕掛かり一覧
  • 進捗率別の分布
  • 長期仕掛かりアラート
長期仕掛かりに注意

出来高が計上されているのに請求が遅れている案件は、資金繰りに影響します。速やかに請求処理を行ってください。

アラート機能

経営上注意が必要な状況を自動的に検知し、アラートを表示します。

アラート種別 条件 重要度
入金遅延 支払期日を過ぎた未入金請求 高
粗利赤字 実績原価が受注金額を超過 高
予算超過 実績原価が予算を超過 中
粗利率低下 粗利率が目標を下回る 中
注文書未受領 口頭受注後14日以上未受領 低
長期仕掛かり 30日以上請求されていない出来高 低

期間・フィルター

表示期間の切り替え

  • 今月 - 当月のデータを表示
  • 前月 - 前月のデータを表示
  • 今年度 - 年度累計を表示
  • カスタム - 任意の期間を指定

フィルター

  • 担当者別 - 営業担当、施工担当で絞り込み
  • 顧客別 - 特定の顧客のみ表示
  • 案件種別 - 通常案件、その他で絞り込み
