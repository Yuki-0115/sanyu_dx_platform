請求・入金管理

請求書発行から入金確認までの債権管理

請求・入金管理とは

工事完了後（または出来高に応じて）請求書を発行し、入金を管理します。未入金の追跡、入金遅延アラートにより、債権管理を効率化します。

工事完了
→
請求書作成
→
請求書発行
→
入金確認
→
消込

請求書の作成

作成手順

  1. 案件詳細画面から「請求」タブを選択 請求一覧が表示されます。
  2. 「新規請求書」ボタンをクリック 請求書作成フォームが表示されます。
  3. 請求基本情報を入力 請求番号（自動）、請求日、支払期日を入力。
  4. 請求明細を追加 見積項目を参照、または手入力で明細を追加。
  5. 金額を確認 小計、消費税、合計金額を確認。
  6. 保存・発行 下書き保存または発行処理。

請求項目

項目 必須 説明
請求番号 - 自動採番（INV + 年月 + 連番）
請求日 ◯ 請求書の発行日
支払期日 ◯ 入金期限（顧客の支払条件に基づく）
請求金額 ◯ 税抜金額
消費税 ◯ 消費税額（10%）
合計金額 - 自動計算（税込合計）

請求パターン

一括請求

工事完了後に全額を一括で請求します。

  • 小規模工事向け
  • 資金繰りに注意

着手金＋残金

着工時に一部（30〜50%）、完了時に残金を請求。

  • 一般的な中規模工事
  • 資金繰りの安定化

出来高請求

月次の進捗に応じて分割請求。

  • 大規模・長期工事
  • 月次出来高の記録が必要
出来高請求の計算

出来高請求額 = 受注金額 × 進捗率 - 既請求額

請求ステータス

ステータス 説明 入金状況
draft 下書き - 未発行 -
issued 発行済 - 顧客に送付済み 未入金
partially_paid 一部入金済 残高あり
paid 入金完了 全額入金
overdue 入金遅延 期日超過・未入金

入金の記録

入金登録手順

  1. 請求詳細画面を開く 請求一覧から対象の請求書を選択。
  2. 「入金登録」ボタンをクリック 入金登録フォームが表示されます。
  3. 入金情報を入力 入金日、入金額を入力。
  4. 「登録」ボタンで保存 入金が記録され、請求残高が更新されます。

入金項目

項目 必須 説明
入金日 ◯ 実際に入金があった日付
入金額 ◯ 入金された金額
備考 - メモ（振込名義が異なる場合など）
分割入金について

入金が複数回に分かれる場合は、その都度入金登録を行います。すべての入金が完了すると、請求ステータスが「入金完了」に変更されます。

入金遅延管理

支払期日を過ぎた未入金請求を管理し、督促を行います。

遅延アラート

遅延期間 アラート 推奨アクション
1〜7日 軽度遅延 確認の連絡
8〜14日 中度遅延 督促状の送付
15日以上 重度遅延 上長への報告・対応協議

ダッシュボードでの確認

入金遅延がある場合、経営ダッシュボードにアラートが表示されます。

  • 遅延件数と合計金額
  • 顧客別の遅延状況
  • 遅延期間の長い順にリスト表示

債権管理レポート

確認できる情報

未入金一覧

発行済み・未入金の請求書一覧。顧客別、期日順で確認。

入金予定

今後の入金予定を期日順に表示。資金繰り計画に活用。

顧客別残高

顧客ごとの未入金残高。与信管理に活用。

入金履歴

過去の入金記録。消込状況の確認。
